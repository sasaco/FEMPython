# 計算結果検証

## 1. sampleBeam\*.fem

一端固定の梁

- 長さL=10
- 幅B=2
- 高さH=1
- ヤング率E=21000
- 荷重F=100

### Z方向たわみ

|  材力理論  | Tetra1  | Wedge1  | Hexa1  | Tetra2  | Wedge2  | Hexa2  | Hexa1非適合  |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| 9.524 | 6.446 | 9.093 | 8.528 | 9.457 | 9.468 | 9.463 | 9.424 |

>４面体１次要素は理論より硬過ぎ、６面体要素でもせん断ロッキングの影響が見られる。
<br>
２次要素および非適合要素では理論解に近い値が得られる。


## 2. sampleThickBeam\*.fem

**1.** より太い梁

- 長さL=10
- 幅B=2
- 高さH=5
- ヤング率E=21000
- 荷重F=100

### Z方向たわみ

| 材力理論 | Tetra1 | Wedge1 | Hexa1 | Tetra2 | Wedge2 | Hexa2 | Hexa1非適合 |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| 0.0762 | 0.0778 | 0.0884 | 0.0875 | 0.0905 | 0.0909 | 0.0909 | 0.0894 |

> ①と違い梁が太いため材力理論との乖離が見られる。<br>
要素の傾向は①と同様だがアスペクト比が正方形に近いためせん断ロッキングの影響はやや小さい。（高さ方向分割数ではないと思う。）<br>
その代わり非適合要素と２次要素の結果にやや差が見られる。<br>
（Wilson-Taylorの非適合要素には曲げ以外のせん断などに補正がかからないためか。）


## 3. sampleBend\*.fem

屈曲した梁

- X方向サイズ100
- Y方向サイズ70
- 梁の幅B=20
- 高さH=15
- ヤング率E=21000
- 荷重F=100

### Z方向たわみ
| 材力理論 | Tetra1 | Wedge1 | Hexa1 | Tetra2 | Wedge2 | Hexa2 | Hexa1非適合 | （参考）BEBar | TBar | Tri1 | Quad1 |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| 0.5943 | 0.3952 | 0.5148 | 0.5590 | 0.5950 | 0.5968 | 0.5964 | 0.5756 | 0.5926 | 0.5990 | 0.4380 | 0.5887 |

> 材力理論は曲げ（X方向梁＋Y方向梁）＋X方向棒の捩れを大胆に仮定したものでラフなものだが、２次要素は理論解に近い値となっている。<br>
せん断ロッキングの影響は非適合要素で小さくなっているものの、曲げ以外のモードもあり完全には無くなっていない。<br>
また梁要素（ベルヌーイ=オイラー梁，ティモシェンコ梁）は理論値に近い値となるが、シェル要素は薄板要素でなくとも若干硬めになる。
